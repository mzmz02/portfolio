# Dependency
HTML/CSS/jQueryを用いている。

# 工夫したところ頑張ったところ
Top pageでプロフィールの部分を写真と文章の2つに分けて、見やすいようにかつ目立つようにした。<br>
見やすいように、使用する色を少なくした。その中でもオリジナリティを出すために、背景の色を2色使うようにした。
メディアクエリを利用して、画面サイズによるレイアウトの崩れを防止した。その際に、レイアウトを考えてフォントサイズや画像のサイズを変更するだけでなく、画面のサイズによってメニューバーを追加した。<br>

WorksのページでWebサイトの画像をクリックすると詳細を表示できるようにして、ポップアップ以外の部分をクリックすることで閉じるようにした。<br>
画像をクリックでポップアップが表示されることや、そのポップアップの中の画像クリックでサイトに飛べることをわかりやすいようにした。
